证明：全微分定义式、[[矩阵的乘法]]结合律证明



$$
dz = \frac{ \partial z }{ \partial u } du + \frac{ \partial z }{ \partial v } dv
= \frac{ \partial z }{ \partial u } \left( \frac{ \partial u }{ \partial x } dx+ \frac{ \partial u }{ \partial y } dy \right)+\dots
$$
